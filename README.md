# Guess That Color Game

### Live Demo: https://qcollinscode.github.io/guess-that-color-game.html

## Installation

Git clone this repository:

```
git clone https://github.com/qcollinscode/ProjectJavascript-ColorGame.git
```

Change directory into the cloned repository:

```
cd ProjectJavascript-ColorGame
```

Install the dependencies that are listed in the package.json file:

```
npm install
```

## Running

Start the application:

```
npm start
```
